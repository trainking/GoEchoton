## 命名约束

## 提交约束

```
type:[(scope)] subject
// 空行
[body]
// 空行
[Footer]
```

### type

|类型|类别|说明|
|:--|--|--|
|feat|代码类|新增功能|
|fix|代码类|bug修复|
|perf|代码类|提高代码性能变更|
|style|代码类|格式化代码，美化代码|
|merge|代码类|合并代码，解决冲突|
|refator|代码类|其他代码变更，口袋选项，不属于上述选项时，选择refator|
|test|非代码类|提交测试代码，单元测试案例等|
|ci|非代码类|部署内容的相关改动，如修改部署文件，ci配置文件|
|docs|非代码类|文档变更|
|chore|非代码类|其他变更，口袋选项，不属于上述选项时，选择chore变更|
